# 🎮 Happy Birthday - Level 18 Game

## Sobre o Projeto

Este é um jogo interativo de aniversário desenvolvido especialmente para celebrar os **18 anos** de forma épica e divertida! O jogador precisa coletar **18 estrelas douradas** para desbloquear uma mensagem de parabéns personalizada, tudo com uma temática gamer que transforma a data especial em uma verdadeira jornada de evolução e conquistas.

---

## 🎯 Como Jogar

1. **Clique nas estrelas douradas** que aparecem no campo de jogo
2. Colete **18 estrelas** para completar a missão
3. Ao coletar todas, uma **tela de vitória** será exibida com uma mensagem personalizada
4. Clique na imagem para abrir em **tela cheia**
5. Use o botão **"Jogar Novamente"** para reiniciar a aventura

---

## ⭐ Características

- **Temática Gamer** - Design inspirado em jogos, com HUD, níveis e XP
- **18 Estrelas para Coletar** - Cada estrela representa um ano de vida
- **Cronômetro** - Desafie-se a completar a missão no menor tempo possível
- **Sistema de Level Up** - Ao atingir 18 estrelas, o jogador avança para o próximo nível
- **Tela de Vitória** - Imagem personalizada com mensagem de aniversário
- **Modo Tela Cheia** - Clique na imagem para visualizar em tela cheia
- **Responsivo** - Funciona perfeitamente em PC, tablet e celular
- **Rejogabilidade** - Botão "Jogar Novamente" para repetir a experiência

---
